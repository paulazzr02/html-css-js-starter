Three Column Layout

세 컬럼 레이아웃은 좌측 사이드바, 메인 콘텐츠, 우측 사이드바로 구성됩니다.

메인 콘텐츠

이 레이아웃은 뉴스 사이트, 포털 사이트, 대시보드 등에서 많이 사용됩니다. 좌측에는 주 네비게이션, 우측에는 관련 콘텐츠나 광고를 배치합니다.

세 컬럼 레이아웃의 특징:

  • 좌측: 주 네비게이션 메뉴
  • 중앙: 주요 콘텐츠
  • 우측: 부가 정보, 위젯, 광고

콘텐츠 섹션 2

메인 콘텐츠 영역은 중앙에 배치되어 시각적 균형을 유지합니다. 좌우 사이드바는 보조적인 역할을 하며, 메인 콘텐츠가 시각적 중심이 됩니다.

콘텐츠 섹션 3

반응형 디자인에서는 작은 화면에서 사이드바들이 접히거나 하단으로 이동하여 단일 컬럼 레이아웃으로 전환됩니다.
